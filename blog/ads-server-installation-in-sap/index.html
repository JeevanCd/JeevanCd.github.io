ADS Server installation check in SAP

written by Jeevan Chand on

The first Steps while developing Adobe forms is to check which ADS server is installed and what is the version of the ADS installed in sap.

To check weather the system has ADS installed or not we can run the various standard programs below.

FP_PDF_TEST_00

This program can be used to check the ADS installed version. If the program gives an error. It means that you SAP system has some connection problems with the ADS Server and you might need help of Basis member.

This above screen shows the successful ADS connection with SAP system.

FP_CHECK_DESTINATION_SERVICE Used to check the Destination service to ADS.

Successful data transfer

FP_CHECK_HTTP_DATA_TRANSFER The above program can be used to can also be used to check the http data transfer

FP_TEST_00 This program is can used to test the pdf file generation.
